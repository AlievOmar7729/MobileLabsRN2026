## Лабораторна робота №1

### Тема: Використання Expo для створення найпростішого додатку React

#### Встановлення:

`npx create-expo-app@latest --template`

√ Choose a template: » Blank

√ What is your app named? ... my-app

#### Проблема з сумісностю версій в Expo Go на телефоні :

Проблема була пов'язана з тим, що на телефоні остання версія програми була застаріла,
ніж та, яка завантажувалась при створенні проекту.

Тому змінив версію expo в package.json - "expo": "~54.0.6"

#### Реалізований функціонал:

Для реалізації застосунку використано `React Navigation` з нижньою вкладочною навігацією (`createBottomTabNavigator`), `SafeAreaProvider` для коректного відображення в межах безпечної області екрана та іконки `Ionicons` для вкладок і карток новин.
Головний екран побудований на основі компонентів `ScrollView`, `View` і `Text`, де новини відображаються у вигляді окремих карток. Фотогалерея реалізована через `ScrollView`, `Image`, `Dimensions` і адаптивний розрахунок розміру плиток для двоколонкового відображення фото. Форма профілю створена за допомогою `TextInput`, `TouchableOpacity` та `useState`, а для перевірки введених даних додано базову валідацію полів.

#### Запуск :

`cd my-app`

`npm start`
